# shahzaibbuilds.me

Personal portfolio and business website for Shahzaib Hassan — AI Automation Engineer based in Lahore, Pakistan.

## What this site does

A Next.js marketing site that showcases AI automation services and captures leads. It includes:

- **Hero section** — intro and value proposition (saves businesses 20+ hours/week)
- **ROI Calculator** — interactive tool for visitors to estimate time/cost savings from AI automation
- **Portfolio** — past projects and case studies
- **Services** — breakdown of offered services (Voice Agents, n8n automation, Python AI, consulting)
- **Pricing** — service tiers and packages
- **Social proof** — testimonials and credibility signals
- **AI Chatbot** — custom chatbot powered by an n8n webhook
- **Lead capture** — exit intent modal and email gate modal
- **Sticky CTA** — persistent call-to-action button
- **SEO** — structured schema.org data (Person, ProfessionalService, FAQPage) for Google rich results

## Tech stack

- **Framework:** Next.js 13 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **UI components:** Radix UI
- **Deployment:** Netlify
- **Analytics:** Vercel Analytics

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
