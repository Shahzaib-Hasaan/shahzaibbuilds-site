# Archived Docs

These documents were written when the site was framed as a **B2B sales funnel** targeting US/EU startup founders. That framing was retired in favor of the current direction: a personal presence hub for credibility, scholarship applications, and professional identity.

The funnel-era code these docs describe (email gating, exit-intent modals, lead-magnet PDF, pricing page, etc.) has been removed from the codebase. These files are kept purely for historical reference — **do not use them as a guide for current work**. For current direction, see [`CLAUDE.md`](../../CLAUDE.md) and [`context/`](../../context/).
